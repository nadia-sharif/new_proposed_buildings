# new_proposed_buildings
WebGIS using MapBox to display new and proposed buildings in the City of Melbourne

## Melbourne Skyline: New and Proposed Buildings
An interactive 3D mapping application that tracks urban development across the City of Melbourne. 
It visualises newly completed, approved, under-construction, and applied-for structures in an immersive 3D view.Users can toggle filters to see buildings by their development status and click on any individual structure to view its detailed metrics in a popup box.

## Core Features
Interactive 3D View: Extruded building geometries representing real-world physical heights.
Status Filtering: Custom UI controls to toggle visibility between different construction phases.
On-Click Data Popups: Dynamic metadata cards pulled directly from the map layers.
Minimalist UI: A clean, flat user interface built from scratch without bulky frontend frameworks.

## Tech Stack
### Frontend Architecture: HTML5, Vanilla JavaScript (ES6+), and Custom CSS3.
### Mapping Engine: Mapbox GL JS v3.
### Data Source: City of Melbourne Open Data Team (2024 Dataset).
